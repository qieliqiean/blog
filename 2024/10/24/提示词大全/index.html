pycharm小技巧

搜索

1. “上帝视角”：万能搜索 (Search Everywhere)

如果你只记一个快捷键，必须是这个。

  • 快捷键： Double Shift (连续按两次 Shift)
  • 作用： 这是一个聚合搜索。它能搜文件名、类名、方法名、甚至 IDE 的设置选项和功能名称。
  • 技巧：
    • 勾选右上角的 "Include non-project items" 可以搜索外部库（site-packages）里的源码。
    • 支持缩写匹配（比如搜 UserProfile，输入 UP 或 UsePro 就能搜到）。

2. 文本内容搜索 (Text Search)

这是最传统的“找字”功能，分为局部和全局。

场景 快捷键 关键功能点
当前文件搜索 Ctrl + F 就在当前打开的文件里找。按 F3 下一个，Shift + F3 上一个。
当前文件替换 Ctrl + R 替换当前文件里的内容。支持正则 (Regex)。
全局搜索 Ctrl + Shift + F 核心技巧： 1. File Mask: 右上角可以设置只搜 *.py 或 *.html，排除杂项。 2. Scope: 可以指定只搜“Project Files”还是整个“Project and Libraries”。
全局替换 Ctrl + Shift + R 慎用！威力巨大，建议先用搜索确认范围，再进行替换。

💡 搜索结果预览窗口 (Find Window):

使用全局搜索时，建议点击右下角的 "Open in Find Window"。这样结果会固定在底部栏，你可以用键盘上下键快速在不同文件的结果间跳转，不用反复按搜索键。

3. 结构导航搜索 (Navigation Search)

当你不想搜具体的代码内容，而是记得文件名字或者类名字时，用这就对了。

  • 搜类 (Class): Ctrl + N
    • 专门用来找 Python 的 class 定义。
  • 搜文件 (File): Ctrl + Shift + N
    • 找具体的 .py, .yaml, .txt 文件。
    • 技巧： 支持路径匹配。比如你想找 app/models/user.py，可以直接输入 app/user 或者 mo/us，PyCharm 能智能匹配到。
  • 搜符号 (Symbol): Ctrl + Alt + Shift + N
    • 直接找某个具体的函数名或全局变量名，不管它在哪个文件里。

4. 逻辑关系搜索 (Usage Search)

这是写代码时最“懂你”的搜索，用来分析代码的影响范围。

  • 查找引用 (Find Usages): Alt + F7
    • 场景： 你想改一个函数名，但不知道它在项目里被哪些地方调用了。
    • 光标放在函数/变量名上，按 Alt + F7，底部会列出所有用到它的地方。
  • 高亮引用 (Highlight Usages): Ctrl + Shift + F7
    • 场景： 只想看当前文件里这个变量在哪里用过。
    • 按完后，当前文件里所有该变量都会高亮显示，按 Esc 取消。

5. 救命稻草：查找动作 (Find Action)

  • 快捷键： Ctrl + Shift + A
  • 作用： 当你忘了某个功能的快捷键，或者找不到某个菜单在哪时用。
  • 例子：
    • 想调整字体大小？按 Ctrl+Shift+A，搜 font size。
    • 想打开 Git 提交界面？搜 commit。
    • 想把代码转成大写？搜 upper case。

总结建议

  1. 日常开发最常用：Double Shift (随便找) 和 Ctrl + Click (鼠标左键点击跳转定义)。
  2. 重构代码必用：Alt + F7 (看哪里调用了)。
  3. 排查 Bug 必用：Ctrl + Shift + F (全局搜报错信息或关键词)。
